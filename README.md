# emacs-dwm

Tiled window manager for Emacs

this package was inspired by [dwm.vim](https://github.com/spolu/dwm.vim)

## commands

- `dwm-load-buffer`
  - Load buffer to main window
- `dwm-next-buffer`
  - Jump to next buffer (clockwise)
- `dwm-prev-buffer`
  - Jump to previous buffer (anti-clockwise)
